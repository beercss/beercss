# Expansion

Expansion contain creation flows and allow lightweight editing of an element.

## Element

```html
<details>
  <summary>...</summary>
  <...>...</...>
</details>
```

## Example

```html
<article>
  <details>
    <summary>
      <h5>Title</h5>
    </summary>
    <p>Description that will be shown after click on summary</p>
  </details>
</article>
```
